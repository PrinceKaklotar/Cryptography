What's in the .hpqc file?

The .hpqc file is a JSON file containing your Base64-encoded ciphertext — this means it opens cleanly in any text editor and uploads reliably across all browsers. Structure:

{
  "format": "HybridPQC-v1",
  "algorithm": "AES-256-GCM",
  "iv": "<base64 nonce>",
  "ciphertext": "<base64 encrypted data + GCM tag>",
  "originalSize": <bytes>
}

The key is NOT stored in the file — it lives only in the key exchange session.

Upload an encrypted .hpqc file to decrypt it. The hybrid key from this session will be used automatically.
